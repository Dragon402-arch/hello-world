## 快捷键

- 加粗 Ctrl + B

- 斜体 Ctrl + I

- 表格 ctrl + T

- 下划线 ctrl+u

- 搜索：ctrl+f 

- 替换：ctrl+h

- 无序列表：输入-之后输入空格 

- 有序列表：输入数字+“.”之后输入空格

- 公式输入不同大小的括号

  ```
  \big [ //比较大的左中括号 
  \bigg [ //再大一点 
  \Big [ //比big大一点 
  \Bigg [ //比bigg大一点
  ```
